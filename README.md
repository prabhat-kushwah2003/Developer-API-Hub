# DevAPI Hub

DevAPI Hub is a full-stack MERN application that allows developers to test APIs, save requests, and explore public APIs like country and weather data.

---

## Features

-  JWT Authentication (Login / Register)
-  API Playground (GET, POST, PUT, DELETE)
-  Save & manage API requests
-  Public API Explorer (Country + Weather)
-  Dynamic Dashboard stats
-  Real-time API responses

---

## Tech Stack

**Frontend**
- React
- Tailwind CSS
- Axios

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB

**Auth**
- JWT + bcrypt
